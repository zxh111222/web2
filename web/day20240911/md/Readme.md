## 关于`域名`和`备案`
- ![img.png](img.png)
- 本质原因：这是我们锅内的“特色”
- 从技术上来说，完全不需要这个环节。有了域名，有了服务器，两者关联上即可
- 解决方法：
  - 去锅外的平台买服务器和域名
  - 买一台符合要求的服务器（要花钱的，不强制要求，自己决定）
  - 不买域名也行，直接用公网IP地址访问
